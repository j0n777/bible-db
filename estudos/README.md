# 📚 Estudos — Arquivo do AGENT Cristão

Este diretório arquiva as perguntas e respostas produzidas em sessões de estudo com o
**AGENT Cristão** (ver [`CLAUDE.md`](../CLAUDE.md) na raiz do repositório), organizadas
por tema em subpastas.

Cada arquivo documenta a pergunta original, a resposta e, quando aplicável, as fontes
históricas/acadêmicas consultadas (fora da própria Bíblia, para questões de história e
arqueologia — no caso de exegese bíblica, a fonte é sempre o texto das Escrituras neste
repositório e os manuscritos originais).
